# Meditation-app
This Meditation App is a simple, browser-based mindfulness timer built using pure HTML, CSS, and JavaScript. It helps users practice meditation with customizable session durations and calming ambient sounds.  The application includes a visual breathing animation, countdown timer, sound selection, and playback controls.

## ğŸ® How It Works

- User selects meditation duration.
- Timer starts countdown using `setInterval`.
- Breathing animation activates using CSS keyframes.
- Ambient sound plays in loop.
- When timer reaches zero:
  - Sound stops
  - Animation stops
  - Completion alert appears
